# 🎓 Student Management System (C++)

## 📌 Project Description
This is a simple Student Management System built using C++.  
It demonstrates Object-Oriented Programming (OOP) concepts such as:
- Classes and Objects
- Encapsulation
- Constructors
- File/Code modularization

---

## 🧠 Features
- Add student information
- Display student details
- Organized using header and source files

---

## 🗂️ Project Structure
src/ → Source files (.cpp)  
include/ → Header files (.h)  
bin/ → Compiled executable  

---

## ⚙️ How to Compile
```bash
g++ src/main.cpp src/Student.cpp -I include -o bin/app
